### 토이 - 개인프로젝트 관리앱

> 김정태
> 

> 프로젝트 기간 : 2022.06.01 ~ 2022.12.06
> 

### 1. 개발 환경

- xcode 14.1
- swift 5.7.0

### 2. 라이브러리

- Firebase/Auth, GoogleSignIn, Firebase/Database
- Toast-Swift
- MaterialComponents/BottomSheet
- DropDown
- lottie-io
- KakaoSDKCommon, KakaoSDKAuth, KakaoSDKUser
